# Code of Conduct

This project follows the [Contributor Covenant, version 2.1][cc-home]. The
full text is available at <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

[cc-home]: https://www.contributor-covenant.org/

## Summary

We pledge to make participation in this project a respectful, harassment-free
experience for everyone, regardless of background or identity.

Expected behavior:
- Be respectful of differing viewpoints and experiences.
- Give and accept constructive feedback gracefully.
- Focus on what is best for the project and the community.
- Show empathy toward other contributors.

Unacceptable behavior includes:
- Personal attacks, insults, or derogatory comments.
- Publishing others' private information without permission.
- Conduct that would be inappropriate in a professional setting.

## Scope

This Code of Conduct applies in all project spaces — the repository,
issue tracker, pull requests, discussions, and any other forum used for
project communication — and also when an individual is representing the
project in public spaces.

## Enforcement

Report violations to **fkulart@mail.ru** with subject prefix `[CONDUCT]`.
All reports will be reviewed and investigated promptly and fairly. The
project maintainer is obligated to respect the privacy and security of
the reporter.

Maintainers who do not follow or enforce this Code of Conduct in good
faith may face temporary or permanent repercussions as determined by
other members of the project's leadership.
